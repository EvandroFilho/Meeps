Visualizar carrinho - UC019

O comprador poderá visualizar os produtos adicionador no carrinho

Ator:

  • Comprador

Pré-condições

  • O comprador deverá estar logado
  • O sistema deve ter produtos cadastrados
  • Terá de haver pelo menos um produto no carrinho

Fluxo Principal

Pós-condições

  • O comprador poderá finalizar a compra, remover um produto selecionado ou voltar a tela de produtos
