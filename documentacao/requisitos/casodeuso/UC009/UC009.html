Finalizar Compra - UC009

O usuário cliente deve poder finalizar uma compra

Ator:

  • Comprador

Pré-condições

  • O usuário deve estar logado
  • O sistema deve ter produtos cadastrados
  • O cliente deve ter produtos no carrinho

Fluxo Principal

Pós-condições

  • A compra deve ser finalizada
