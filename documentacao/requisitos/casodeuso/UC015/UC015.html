Adicionar Foto Do Produto - UC015

O usuário vendedor deve poder adicionar uma foto ao produto

Ator:

  • Vendedor

Pré-condições

  • O usuário deve estar logado
  • O sistema deve ter produtos cadastrados

Fluxo Principal

Pós-condições

  • O produto deve ter uma foto adicionada
