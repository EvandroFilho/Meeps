Adicionar ao Carrinho - UC007

O sistema deve adicionar o produto selecionado pelo usuário ao carrinho de compras

Ator:

  • Cliente

Pré-condições

  • O usuário deve estar logado
  • O sistema deve ter produtos cadastrados
  • O usuário deve ter selecionado um produto

Pós-condições

  • O carrinho comporta o(s) item(ns) selecionado(s) pelo usuário
