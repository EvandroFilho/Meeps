Listar vendedores - UC004

O MEEPS listará os vendedores disponíveis do produto selecionado pelo comprador

Ator:

  • MEEPS

Pré-condições

  • O comprador deverá estar logado
  • O sistema deve ter produtos cadastrados
  • O comprador deverá ter selecionado um produto

Fluxo Principal

Pós-condições

  • O comprador poderá selecionar um vendedor
