Objetivo

Este documento tem como objetivo definir o significado de termos utilizados na documentação

Definições

  • Cliente: Usuário que realiza compras através do sistema
  • Vendedor: Usuário que realiza vendas através do sistema
  • Meeps: sistema que intermedeia a venda de produtos entre cliente e vendedor
